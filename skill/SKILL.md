# 명리 공부 노트 블로그 스킬 (Saju Study Blog)

## 이 스킬이 하는 일

30대 여성 화자가 "명리학을 조금씩 공부하며 정리하는 노트" 톤으로 네이버 블로그 정보성 글을 꾸준히 발행할 수 있도록, 6개의 전문 에이전트가 순차적으로 협업하는 파이프라인을 제공한다.

완전 자동 발행이 목적이 아니다. **"AI가 초안~SEO~검수까지 정성껏 준비 → 사람이 10~15분 최종 손질 → 네이버에 발행"**이 이 스킬의 정상 작동 방식이다. 이 구조여야 네이버 C-Rank·D.I.A. 2.0 알고리즘의 저품질 필터에 걸리지 않는다.

## 블로그 브랜드 정의

**블로그 이름**: 일간일기
**부제**: 사주 일간과 MBTI로 나와 사람을 읽는 법
**한 줄 약속**: MBTI로 잡히지 않던 부분을, 사주 일간이 채워줍니다. 관계도, 나도 한 겹 더 선명해져요.

**타깃 독자 우선순위**:
1. **자기이해형** — "나 이런 사람이었구나"를 찾는 30~40대 여성
2. **관계해석형** — 남친·친구·가족·동료를 이해하고 싶은 독자
3. **공부 관심형** — 명리학을 공부하고 싶은 소수

## 화자 페르소나

- **정체성**: 30대 여성. 몇 년 전부터 명리학을 공부해왔고, MBTI와 겹쳐 보는 해석을 좋아함.
- **포지션**: **"공부한 걸로 풀어주는 사람"**. 전문가라기보단 해석을 나누는 사람. 학습자와 해석자 사이.
- **톤**: 차분한 해요체. 과하게 친근하지도 않고, 너무 건조하지도 않음.
- **글의 태도**: 단정하지 않음. "~라고 해석돼요", "~하는 경향이 있다고 해요"
- **절대 하지 않는 것**: 점쟁이 말투, 단정적 예언, 특정 종교 색, 과한 이모지, "~하세요!" 같은 명령형 권유

## 모든 글의 필수 시그니처 섹션

모든 글에는 **"MBTI로는 ___, 일간으로는 ___"** 섹션이 반드시 들어간다.
이 섹션이 이 블로그의 브랜드 차별화 포인트다. 포스텔러·유튜브 경쟁자에게는 없는 에디터 시그니처.

## 6개 에이전트 파이프라인 개요

```
[1] 키워드 리서치 에이전트
     → 오늘 쓸 글감과 타겟 키워드 결정
[2] 명리 지식 정리 에이전트
     → 정확한 명리학 내용 팩트 정리
[3] 글 작성 에이전트 (공부 노트 톤)
     → 페르소나 톤으로 초안 작성
[4] SEO 최적화 에이전트
     → 제목·소제목·키워드·메타 조정
[5] 리스크 검수 에이전트
     → 단정 표현·법적 리스크 필터링
[6] 발행 준비 에이전트
     → 이미지·링크·발행 체크리스트 생성
      ↓
   사람 최종 검수 (10~15분)
      ↓
   네이버 블로그 발행
```

각 에이전트의 상세 시스템 프롬프트는 `agents/` 폴더에 분리되어 있다. Claude 프로젝트 기능을 쓴다면 프로젝트 6개를 만들어 각각을 세팅하고, Make.com·n8n 같은 노코드 툴이나 Claude Agent SDK를 쓴다면 각 프롬프트를 해당 노드에 주입한다.

```
│   ├── 2-saju-expert.md         # 명리 지식 정리 에이전트
│   ├── 3-writer.md              # 글 작성 에이전트
│   ├── 4-seo-optimizer.md       # SEO 최적화 에이전트
│   ├── 5-risk-reviewer.md       # 리스크 검수 에이전트
│   └── 6-publisher.md           # 발행 준비 에이전트
├── templates/
│   ├── post-structure.md        # 글 구조 템플릿
│   ├── title-patterns.md        # 제목 패턴 모음
│   ├── human-edit-checklist.md  # 사람이 최종 손질할 체크리스트
│   └── 30-post-ideas.md         # 첫 30개 글감 리스트
└── references/

    ├── tone-examples.md         # 톤앤매너 좋은 예/나쁜 예

```

## 이 스킬을 사용하는 절차

### 새 글 1편을 만들 때

1. `agents/1-keyword-researcher.md`를 불러와서 오늘 쓸 주제·키워드를 결정한다.
2. 결정된 키워드를 `agents/2-saju-expert.md`에 넘겨 명리 팩트 정리본을 받는다.
3. 팩트 정리본을 `agents/3-writer.md`에 넘겨 블로그 초안을 받는다.
4. 초안을 `agents/4-seo-optimizer.md`에 넘겨 SEO 완성본을 받는다.
5. SEO 완성본을 `agents/5-risk-reviewer.md`에 넘겨 리스크 필터링된 최종본을 받는다.
6. 최종본을 `agents/6-publisher.md`에 넘겨 이미지 지시문·발행 체크리스트를 받는다.
7. `templates/human-edit-checklist.md`를 보면서 사람이 10~15분 손질 → 네이버 발행.

### 블로그 초기 세팅을 할 때

1. `templates/30-post-ideas.md`를 열어 첫 30개 글감을 확인한다.
2. 주 3편 페이스라면 약 10주치 분량이다. 계절·시즌 이슈에 맞게 순서를 조정한다.
3. `references/tone-examples.md`를 먼저 읽고 화자 톤을 머릿속에 박아둔다.
4. 1주차는 자동화 없이 수동으로 3편을 만들어보며 각 에이전트 프롬프트를 본인 스타일에 맞게 미세 조정한다.

## 3가지 핵심 원칙

### 원칙 1. 단정하지 않는다
명리학은 해석의 영역이다. "~합니다"가 아니라 "~한 경향이 있다고 해요", "~로 해석되는 경우가 많아요"로 쓴다. 이 원칙은 리스크 회피이자 동시에 화자 페르소나(공부하는 사람)의 일관성이다.

### 원칙 2. AI 티를 적극적으로 지운다
아래 표현은 AI 흔적이므로 검수 단계에서 전량 제거한다.
- "결론적으로", "즉,", "~는 매우 중요합니다", "다양한", "흥미롭게도"
- 번호 매김으로 된 과한 리스트업 (3개 이상 연속 리스트는 지양)
- 모든 문단이 비슷한 길이로 가지런한 형태

### 원칙 3. 경험의 빈칸을 남긴다
글 작성 에이전트는 본문 중 1~2군데에 `[내 경험]` 또는 `[공부하며 느낀 점]` 같은 빈칸을 반드시 남긴다. 사람이 이 빈칸을 채우는 10분이 이 블로그의 C-Rank 지수를 만든다.

## 수익화 연결 규칙

- 쿠팡 파트너스 링크는 **글당 1~2개만**. 자연스럽게 본문 맥락에 녹이기.
- 추천 품목: 명리학 입문서, 타로 카드, 풍수 관련 소품, 다이어리·노트, 수정·원석 액세서리
- "구매하세요" 같은 직접 권유 금지. "저는 이 책으로 공부하고 있어요" 같은 경험 기반 언급만.
- 쿠팡 파트너스 이용 시 **의무 문구("이 포스팅은 쿠팡 파트너스 활동의 일환으로 수수료를 제공받습니다")**를 본문 하단에 반드시 삽입. (공정거래위원회 추천·보증 심사지침)

## 다음 단계로 추천

스킬 세팅이 끝나면 이 순서대로 진행하는 걸 추천한다.

1. `templates/30-post-ideas.md`의 1번 글감으로 파이프라인 전체를 한 번 끝까지 돌려본다.
2. 결과물을 읽고 어색한 부분을 각 에이전트 프롬프트에 반영한다.
3. 주 3편 리듬으로 2~3주 운영한다.
4. 프롬프트가 안정되면 Make.com/n8n으로 자동화 연결을 고려한다.
