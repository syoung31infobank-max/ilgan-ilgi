# 네이버 블로그 SEO 실전 규칙

에이전트 4(SEO)가 참조합니다. 2024~2026년 네이버 알고리즘 기준.

## 네이버가 좋은 글을 판단하는 3축

### 1. C-Rank (블로그의 주제 집중도·신뢰도)
- **한 주제에 집중한 블로그**일수록 지수 ↑
- 같은 카테고리 글이 계속 쌓이는 게 핵심
- 댓글·공감·스크랩이 지수에 반영
- 장기 운영이 결정적 (3개월차부터 효과, 6개월차부터 체감)

### 2. D.I.A. 2.0 (글 단위의 정보성·경험성)
- **글 단위**로 깊이와 정보량을 평가
- AI 티 나는 얇은 글을 가장 싫어함
- 체류시간·완독률이 핵심 지표
- 본인 경험·사진이 있는 글을 선호

### 3. 스마트블록 (검색 의도 매칭)
- 사용자가 검색한 의도에 얼마나 맞는 글인지
- 제목·도입부 100자에 키워드 의도가 드러나야 함
- 질문형 검색어에는 답변형 구조로 응답

## 글 단위 실전 규칙

### 제목
- **25~40자** (모바일 검색결과 잘림 방지)
- 메인 키워드는 **앞쪽 1/3**에 배치
- 숫자(3가지, 5가지)는 10편 중 1~2편만
- 특수문자 최소화 (느낌표 1개까지)

### 도입부 (첫 100자)
- 메인 키워드 1회 자연 포함
- 독자의 검색 의도를 그대로 반영하는 한 문장
- "오늘은 ~에 대해 정리해보려고 해요" 같은 자연 도입

### 본문
- **1500자 이상** (정보성 글 최소선)
- **H2 소제목 4~5개**로 구조화
- 메인 키워드 밀도 **1~2%** (강제 삽입 금지)
- 연관 키워드(LSI) 3~5개 자연 녹임
- 문단 길이 변주 (짧은 문단·긴 문단 섞기)

### 이미지
- **3~5장 삽입** 권장
- 첫 이미지는 썸네일 자동 지정 → 가장 공들이기
- 모든 이미지에 **대체 텍스트(alt)** 설정
- alt에 메인 키워드 자연 포함 (1장 정도)

### 태그
- **7~12개** 적정 (30개까지 가능하나 많을수록 지수 분산)
- 메인 키워드 + 서브 키워드 + 블로그 카테고리

### 발행 후
- 24시간 내 색인 여부 확인 (블로그명+제목으로 검색)
- 댓글이 달리면 반드시 답글 (체류·참여 신호)

## 피해야 할 SEO 안티패턴

- 같은 키워드 5회 이상 반복 삽입
- 키워드만 나열한 제목
- 1000자 미만의 얇은 글
- 이미지만 많고 본문 적은 글
- 매일 동일 구조·동일 어투의 양산 글
- 외부 링크 과다 삽입
- AI 생성 글을 사람 손 거치지 않고 바로 발행

## 기대 곡선 (현실적 수치)

- **1~2개월차**: 거의 유입 없음. 네이버가 블로그를 학습하는 기간.
- **3개월차**: 일부 롱테일 키워드에서 상위 노출 시작. 일 방문자 50~100명.
- **6개월차**: 주제 응집도가 쌓이며 검색 유입 본격화. 일 방문자 300~500명.
- **12개월차**: 시즌 키워드에서 1페이지 상위 노출. 월 10~30만 원대 수익 가능 구간.

이 곡선은 **주 3편 이상 꾸준히 발행 + 사람 손 거친 글**이라는 조건이 유지될 때예요.
