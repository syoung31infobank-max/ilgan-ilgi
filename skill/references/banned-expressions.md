# 쓰지 말아야 할 표현 목록

에이전트 3(작성)과 에이전트 5(검수)가 참조합니다.

## 1. 단정·예언형 (즉시 수정)

| 금지 | 대체 |
|---|---|
| ~한다 / ~할 것이다 | ~한 경향이 있다고 해요 |
| 당신은 ~ | (2인칭 단정 자체를 삭제) |
| 무조건 ~ | (삭제) |
| 반드시 ~ | (삭제) |
| 이 사주는 ~다 | 이 사주는 ~로 해석되는 경우가 많아요 |
| 결혼 못 한다 | 결혼 시기가 늦어질 수 있다고 해요 |
| 이혼한다 | 부부 관계에 긴장이 올 수 있다고 해석돼요 |
| 부자 된다 | 재물운이 열리는 시기로 해석되기도 해요 |
