# 사람이 최종 손질할 체크리스트 (10~15분)

AI 파이프라인이 아무리 잘 돌아도, 이 10~15분을 건너뛰면 블로그가 저품질 처리돼요.
이 체크리스트는 '지수를 쌓는 마지막 공정'이라고 생각해주세요.

## 1. 경험 빈칸 채우기 (가장 중요, 5분)

- [ ] 본문 중 `[내 경험]` 자리에 본인 이야기 2~3문장 작성
- [ ] 공부하며 실제로 막혔던 지점, 혹은 깨달은 지점
- [ ] "책에서 ~라고 봤는데 저는 처음에 ~로 이해했어요" 같은 진짜 감상

**이 2~3문장이 네이버 C-Rank에 가장 크게 기여해요.** 이게 없으면 AI 양산 글과 구분이 안 돼요.

## 2. 본인 사진 1장 삽입 (3분)

아래 중 하나라도 좋아요.

- [ ] 공부하던 책 펴놓은 사진
- [ ] 노트에 필기한 사진 (얼굴 안 나와도 됨)
- [ ] 커피잔 + 책 구도
- [ ] 손글씨로 쓴 오행·십성 한자 메모

**"실제 사람이 쓴 글"이라는 신호가 가장 강력해요.**

## 3. 문장 결 최종 점검 (3분)

소리 내서 한 번 읽어봐요. 어색한 부분을 고칩니다.

- [ ] 내 평소 말투와 다른 문장 없나
- [ ] 딱딱하거나 번역 투 문장 없나
- [ ] 같은 단어가 연달아 반복되지 않나
- [ ] 이모지 혹시 과하게 들어갔나

## 4. 쿠팡 파트너스 처리 (2분)

- [ ] 쿠팡 파트너스 링크 생성했나 (본인 계정)
- [ ] 본문에 자연스럽게 삽입했나 (1~2곳만)
- [ ] 본문 하단에 의무 고지 문구 삽입했나
  > "이 포스팅은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다."

## 5. 발행 설정 (2분)

- [ ] 카테고리 맞게 선택
- [ ] 태그 7~12개 (에이전트 6 추천 기준)
- [ ] 공개 설정: 전체공개 / 검색 허용
- [ ] 발행 시간: 평일 저녁 8~10시 혹은 오전 8~10시

## 6. 발행 후 (1분)

- [ ] 본인 블로그 검색으로 해당 키워드 검색해보기 (24시간 내 색인 여부 확인)
- [ ] 글감 대장에 "발행 완료" 체크, 다음 글에 내부 링크로 연결할 후보인지 기록

---

## 매일이 아니라 '주 1회 루틴'으로 묶는 걸 추천

- **일요일 저녁 30분**: 에이전트 1로 이번 주 3편 글감 확정
- **월/수/금 각 30~40분**: 에이전트 2~6 파이프라인 돌리고, 10~15분 사람 손질 후 발행

주당 총 투입 시간: 약 **2시간~2시간 30분**. 이 정도면 직장 다니면서도 안정적으로 돌릴 수 있어요.
